# CV Template Generator

Editor de currículo com preview em tempo real e exportação para PDF.

[Acesse online](https://gabrielmatozo.github.io/cv-template-generator/)

## Stack

HTML, CSS e JS puro. Sem dependências. Compatível com GitHub Pages.

## Recursos

- Preview em tempo real
- Exportação para PDF (A4)
- Suporte para português, inglês e espanhol
- Salva dados automaticamente no navegador
- Controle de visibilidade por seção

## Estrutura

```
index.html
assets/
  favicon.svg
  css/styles.css
  js/app.js
```

## Uso local

```bash
npx http-server -p 8888
```
